Menus e Botões


  Como todo software, a Enciclopédia é manejada mediante Digitação, uso do mouse, Botões e Menus.


Abaixo temos a imagem dos Menus e botões (faltando alguns botões que, não obstante, iremos relacionar abaixo).



Explicamos a seguir estes itens.


1. Menus

Menus são elencos de funções das quais o usuário pode servir-se, ficando acima, no programa, separados por títulos que lhes definem o gênero.
(são acessados pelo mouse ou se utilizando a tecla Alt junto com a letra sublinhada em cada um deles)

São os seguintes:


a) LexMaximus - Contém os seguintes itens:


Sobre - dados dos editores, obra e analista de sistemas bem como a ficha técnica do programa.

Licença de uso - Por lei, todo software é objeto de licença de uso elaborada pelos editores, sendo um contrato de adesão, que inclusive é aceito pelo usuário antes de instalar o programa, nele prescrevendo-se os termos de uso, a responsabilidade e o prazo de garantia do produto.
Página Inicial - Exibe, quando há conexão com a internet uma página com dicas ou notícias da Enciclopédia.

Notas - Notas escritas pelo usuário.
Índice - Volta à lista principal quando a lista de notas ou a lista histórico das consultas está sendo exibida
Sair - Função para fechar o programa.


 

b) Configurações - Contém o item configurar fontes e skins, relativo à configuração das letras da lista e dos verbetes, cor da janela e estilos da janela.


c) Editar - Contém os seguintes itens:


Selecionar tudo - seleciona todo texto dos verbetes

Copiar - Copia o texto selecionado Teclas de atalho: Crtl+C
Imprimir - Imprime o verbete
Voltar histórico - retrocede os verbetes já consultados. Tecla de atalho: F11
Avançar histórico - refaz os passos dos verbetes consultados após usar a função de retroceder. Tecla de atalho: F12

 

d) Ajuda - Contém o item relativo ao Socorro, Ajuda de explicação do programa.

 

2. Botões

São ícones que,uma vez pressionados ou selecionados, acionam uma função. São estes os seguintes:

Botão para voltar à lista principal. Pode ser usado também, no lugar deste botão, o recurso de pressionar qualquer tecla para voltar à lista principal. Aparecerá ativo quando estiver selecionada a lista de notas ou a lista do histórico das consultas.
Botão que exibe o histórico das consultas realizadas naquela sessão. Ele passa a ficar ativo somente após ser acessado o primeiro verbete. Vide o tópico Histórico para maiores explicações.
Botão de impressão do texto. Vide o tópico copiar e imprimir.
Botão da lista de notas. Ele passa a ficar ativo somente após ser inserida a primeira nota pelo usuário. Vide o tópico Notas para maiores explicações.
Botão de cópia do texto selecionado. Vide o tópico copiar e imprimir.
Botões para avançar e retroceder no pelas expressões selecionadas como resultado de uma consulta. Vide o tópico Consulta para maiores explicações.
Botões para avançar e retroceder pelas normas já consultados. de cópia do texto selecionado. Vide o tópico Histórico para maiores explicações.
 Botões Layout. Redimensionam a exibição da tela sob várias formas. Vide o tópico Layout para maiores explicações.

Botões de seleção (ao ticar fica preenchido em vermelho como na imagem) das modalidades de consulta digitada. A pesquisa pode ser acionada pela tecla Enter ou o botão buscar. Vide o tópico Consulta para maiores explicações.
